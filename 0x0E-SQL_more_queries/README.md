- [x] What’s a PRIMARY KEY
- [x] What’s a FOREIGN KEY
- [x] How to use NOT NULL and UNIQUE constraints
- [x] How to retrieve datas from multiple tables in one request
- [x] What are subqueries
- [x] What are JOIN and UNION
